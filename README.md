## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```
   git clone https://github.com/Paymanshus/HeartRate-Reader
   ```
2. Install requirements from requirements.txt

   ```
   pip install -r requirements.txt
   ```

3. Run the Flask app
   ```
   python app.py
   ```
